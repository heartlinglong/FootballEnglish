# Web应用PRD：「足球英语小镇」教育游戏（完整版）

## 一、产品概述

### 1.1 产品名称
足球英语小镇（Football English Town）

### 1.2 产品定位
面向6-10岁儿童的教育类Web游戏，通过"足球闯关学英语+创意搭房子"的趣味玩法，融合英语单词/短句学习与空间思维培养，让儿童在游戏中主动探索、快乐学习。

### 1.3 核心目标
- 帮助儿童通过语音互动掌握基础英语词汇（覆盖用户日记中提到的高频词，如动物、物品、自然现象等）
- 培养空间认知能力与动手创造力
- 降低学习压力，提升英语学习兴趣，同时满足家长对学习效果的追踪需求

### 1.4 核心价值
- **趣味化学习**：将英语单词学习融入足球游戏，用"踢飞单词""进球通关"的互动形式替代传统背诵
- **即时反馈**：语音识别实时响应，搭配动画、音效增强成就感
- **创意释放**：搭房子环节作为奖励，让儿童在学习后通过自由搭建放松并培养创造力
- **家长可控**：提供学习数据追踪与使用时长管理，平衡学习与娱乐

---

## 二、目标用户

### 2.1 核心用户
- **年龄**：6-10岁儿童（小学1-3年级）
- **特征**：喜欢互动游戏，对动画、音效敏感，处于英语启蒙阶段，注意力持续时间较短
- **需求**：通过游戏化方式学习英语，获得成就感，同时希望有自由创造的空间

### 2.2 次要用户
- **家长**（儿童监护人）
- **需求**：了解儿童学习进度（如掌握单词量、正确率），控制游戏时长，确保内容安全（无不良信息）

## 三、核心功能模块

### 模块1：足球英语闯关（核心学习环节）

#### 3.1.1 关卡设计

**难度分级**：按英语水平分为3级，每级含5个小关卡，逐步提升复杂度：

| 等级 | 关卡 | 内容 |
|------|------|------|
| 初级 | L1-L5 | 基础单词（3-5个/关卡） |
| 中级 | L6-L10 | 短语/短句（5-8个/关卡） |
| 高级 | L11-L15 | 简单对话（8-10个/关卡） |

> 💡 以下单词为初始示例，实际运行时所有单词均从**飞书多维表格在线词库**动态拉取（详见模块5），支持随时扩充和调整，无需修改代码。

**初级单词分类（示例）**：
- **动物类**：panda（熊猫）、goose（小鹅）、dog（小狗）
- **物品类**：notebook（计划本）、sofa（沙发）、car（小汽车）、bus（公交车）、taxi（出租车）、plane（飞机）、train（火车）、subway（地铁）
- **自然现象/其他**：snow（雪）、Ultraman（奥特曼，注：音译专有名词，标注"卡通角色名"）

**中级示例**：
- "a small dog"
- "red sofa"
- "take the bus"

**高级示例**：
- "What's this? It's a panda."
- "Where is the bus?"

**场景与视觉**：
- 主场景：卡通足球场（绿色草地、白色球门、彩色单词泡泡），背景可切换（晴天/雨天/星空，每通关3关解锁1个新背景）
- 角色：默认提供3个卡通形象（小熊、小兔、小猫），支持通过"创意徽章"解锁更多角色（如小恐龙、机器人）

#### 3.1.2 核心玩法

**角色与足球控制**：
- 操作方式：电脑端用方向键（↑↓←→）控制角色跑动，移动端用屏幕虚拟摇杆
- 足球跟随：角色带球时，足球始终在角色前方1-2步距离，不可脱离（避免"丢球"挫败感）

**单词障碍互动**：
- 触发机制：角色靠近悬浮的"单词泡泡"（直径约50px，彩色半透明）时，泡泡放大并播放提示音（如"请说出这个单词！"）
- 语音输入：玩家通过麦克风说出单词（支持中文提示，如"熊猫的英语是？"），系统调用**火山引擎（Volcengine）流式语音识别接口**实时判断发音准度。
- 结果反馈：
  - ✅ 正确：单词泡泡爆炸（伴随"嘭"音效+碎片动画），角色欢呼（如"太棒啦！"），同时屏幕右侧"能量条"+10%（能量条满后角色可短暂加速）
  - ❌ 错误/超时（5秒未输入）：泡泡闪烁红色，播放"再试试！"提示音，显示中文释义（如"熊猫→panda"），3次错误后出现"跳过"按钮（点击后泡泡消失，但能量条-5%）

**进球判定**：
- 通关条件：踢飞当前关卡所有单词泡泡后，球门解锁（从灰色变为金色发光），角色需移动至球门前3步内
- 射门操作：点击屏幕"射门"按钮（或语音说"shoot"），足球沿直线飞向球门，进球后播放烟花动画+"恭喜通关！"语音，自动进入"搭房子"环节

#### 3.1.3 进度与激励

- **关卡进度**：屏幕顶部显示"已踢飞单词数/总单词数"（如"3/5"），底部显示能量条
- **成就系统**：每通关1关获得1颗"星星"，集齐5颗星星解锁1个新角色/背景
- **错题复习**：关卡结束后，自动生成"错题本"（仅显示本关错误单词），支持点击重新练习发音

### 模块2：创意搭房子（奖励与创造环节）

#### 3.2.1 触发与入口
- 完成1个足球闯关关卡后自动弹出"搭房子"入口（如"恭喜！现在可以去搭房子啦～"），也可在主界面点击"创意工坊"进入
- 单次开放时长：5-10分钟（可手动提前退出），避免过度沉迷

#### 3.2.2 核心玩法

**搭建材料**：
- **基础积木**：按用户描述提供形状（长方形、正方形、椭圆形、三角形）、颜色（红/黄/蓝/绿/橙）、尺寸（大/中/小），每种形状初始解锁3种颜色，后续通过关卡进度解锁更多
- **特殊道具**：通关中级关卡后解锁"装饰积木"（如窗户、门、屋顶、烟囱），增加搭建细节

**操作方式**：
- 拖拽：鼠标/触屏长按积木拖至搭建区域（白色空地，大小约800×600px）
- 旋转/缩放：选中积木后，电脑端用鼠标右键旋转（每次90°）、滚轮缩放（范围50%-200%）；移动端双指缩放、单指旋转
- 删除/撤销：将积木拖入右下角"回收箱"图标删除，或点击"撤销"按钮退回上一步操作

**搭建模式**：
- **自由模式**：无目标限制，玩家可搭建任意造型（房子、城堡、通道等），界面显示"发挥你的创意吧！"提示
- **引导模式**（初级用户）：显示简单轮廓提示（如"搭一个有2面墙+1个屋顶的房子"），完成后弹出"创意徽章"（如"小小建筑师"）

#### 3.2.3 作品保存与分享
搭建完成后，点击"保存作品"生成图片（PNG格式），支持：
- 本地下载（保存至设备相册）
- 分享至家长账号（需家长在"家长中心"开启权限），家长可在手机端查看儿童作品

### 模块3：家长中心（辅助管理功能）

#### 3.3.1 学习数据统计
- **核心数据**：
  - 学习时长（今日/本周累计）
  - 单词掌握情况（总学习单词数、正确率、高频错误单词）
  - 关卡进度（已通关关卡数、当前等级）
- 数据展示：以图表形式呈现（如正确率折线图、单词分类饼图），直观反馈学习效果

#### 3.3.2 功能设置
- **使用控制**：设置每日最长使用时长（默认30分钟，可调整为15-60分钟），超时后自动锁定游戏
- **内容管理**：选择英语难度等级（初级/中级/高级），开启/关闭"错题复习"功能
- **安全设置**：开启"家长密码"（修改设置需验证），关闭"作品分享"功能（避免隐私泄露）

#### 3.3.3 反馈与建议
提供"问题反馈"入口（如"单词发音不清晰""搭房子卡顿"），家长可提交文字/截图，研发团队定期回复。

### 模块4：系统配置与开发者设置（Local-First 特色功能）

#### 3.4.1 API 配置面板
为了支持本地直接运行且保护用户隐私，应用提供统一的 API 配置面板：

**火山引擎（语音识别）**：
- `AppID` (火山引擎应用标识)
- `AccessKey` (身份验证密钥)
- `SecretKey` (安全访问密钥)

**飞书开放平台（在线词库）**：
- `App ID` (飞书自建应用标识)
- `App Secret` (飞书应用密钥)
- `多维表格链接` (词库表格的分享链接或 `app_token`)

- **存储机制**：所有 API 密钥均加密存储于浏览器的 `localStorage` 中，不经过任何第三方服务器，确保数据隐私安全。

#### 3.4.2 连接验证机制
- **验证功能**：点击“验证连接”按钮，系统将通过一次空的语音流初始化请求，验证当前配置的 API 是否可正常联通。
- **反馈状态**：
    - ⏳ **验证中**：显示加载动画。
    - ✅ **成功**：显示“连接成功，语音功能已就绪”。
    - ❌ **失败**：显示报错详情（如“AppID 错误”或“网络超时”），并引导用户检查火山引擎后台配置。

#### 3.4.3 环境检查
- 自动检测设备麦克风权限，若未开启则弹出引导弹窗提示用户授权。
- 检测飞书词库连接状态，确保词库可正常拉取。

### 模块5：飞书多维表格在线词库（动态内容管理）

#### 3.5.1 设计理念
单词内容不硬编码在应用中，而是通过**飞书多维表格**进行在线管理。老师或家长可以随时在飞书表格中增删改单词，应用自动同步最新内容，真正实现"内容与代码分离"。

#### 3.5.2 飞书多维表格结构

| 字段名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| english | 文本 | 英文单词或短语 | panda |
| chinese | 文本 | 中文释义 | 熊猫 |
| level | 单选 | 难度等级（初级/中级/高级） | 初级 |
| category | 单选 | 分类标签 | 动物 |
| phonetic | 文本 | 音标（可选） | /paendae/ |
| example_sentence | 文本 | 例句（中高级用，可选） | It is a panda. |
| enabled | 勾选框 | 是否启用（方便临时禁用某些单词） | true |

#### 3.5.3 数据拉取与缓存策略

- **首次启动**：应用通过飞书开放 API 拉取全部已启用（enabled=true）的词库数据，按难度和分类结构化存储在浏览器 localStorage 中。
- **后续使用**：优先使用本地缓存的词库数据，**无需每次联网**，保证离线也能正常学习。
- **手动同步**：配置面板提供"同步词库"按钮，老师或家长更新表格后点击即可拉取最新数据。
- **同步反馈**：显示"上次同步时间"及词库统计（如"当前词库：初级 42 词 / 中级 28 词 / 高级 15 词"）。

#### 3.5.4 关卡单词抽取逻辑

- 每个关卡开始时，根据当前关卡的**难度等级**，从对应词库中**随机抽取**指定数量的单词（初级 3-5 个，中级 5-8 个，高级 8-10 个）。
- 优先抽取玩家**未学过**或**历史正确率较低**的单词，避免重复练习已掌握的内容。
- 同一关卡内不重复出现相同单词。

## 四、用户交互流程

### 4.1 首次使用流程
1. **引导页**：卡通角色动画演示游戏玩法（"踢单词→进球→搭房子"），配语音讲解
2. **家长设置**：跳转至"家长中心"，选择英语难度（默认初级）、每日时长（默认30分钟），设置家长密码
3. **角色选择**：儿童从3个默认角色中选择1个，进入第一关（L1：动物类单词）

### 4.2 日常使用流程
1. **主界面**：显示已解锁关卡、当前等级、星星数量，点击"开始闯关"进入足球关卡
2. **足球闯关**：控制角色踢单词→进球→解锁搭房子
3. **搭房子**：自由搭建/引导搭建→保存作品→返回主界面
4. **家长查看**：家长登录"家长中心"，查看学习数据、调整设置

## 五、非功能需求

### 5.1 性能要求
- **加载速度**：首次加载≤5秒，关卡切换≤2秒
- **语音识别**：响应时间≤2秒，准确率≥90%（针对6-10岁儿童发音特点采用火山引擎儿童语音模型或通用发音评估模型）
- **并发支持**：支持1000人同时在线，无卡顿/崩溃

### 5.2 兼容性
- **设备**：适配PC（Chrome/Edge/Safari浏览器）、平板（iPad/安卓平板）、触屏笔记本
- **分辨率**：支持1366×768及以上分辨率，移动端自适应屏幕尺寸

### 5.3 安全性
- **数据隐私**：儿童语音数据仅用于实时识别，不存储；学习数据加密存储，仅家长可查看
- **内容安全**：所有单词、图片、动画均通过教育专家审核，无暴力/低俗内容
- **账号安全**：家长账号需手机号验证码登录，支持密码找回

### 5.4 可扩展性
- **单词库更新**：预留接口，支持定期新增单词（如节日主题、季节主题）
- **关卡扩展**：支持新增"主题关卡"（如"海底足球""太空足球"），适配不同学习场景
